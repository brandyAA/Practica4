# Practica4

## Test notificacion push con ntfy.sh

En esta practica, vamos a crear una integracion continua con github actions  en el cual tendremos un programa hola mundo en javascript (o en cualquier otro lenguaje). Al subir el repositorio a la rama main que mande una notificacion a ntfy.sh/devops-itla

Se requiere lo siguiente:
1. Crear un programa hola mundo en cualquier lenguaje
2. Crea el alerta.yml de github actions para que al subir codigo al repositorio main envie una notificacion a ntfy.sh/devops-itla

Valor 5 pts
